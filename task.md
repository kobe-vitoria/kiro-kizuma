Description

Implementar uma automação que analise clusters de tickets repetidos e gere rascunhos de artigos no Confluence, facilitando a documentação proativa de dúvidas frequentes.

Como Gestor de Suporte,

Quero uma automação que agrupe os tickets mais repetidos do mês e crie automaticamente um rascunho de documentação no Confluence,

Para que possamos atacar os maiores gargalos de dúvidas dos clientes rapidamente.

Critérios de Aceite:

[ ] Script de análise de dados rodando mensalmente sobre a base de tickets fechados para identificar clusters de repetição (ex: "como redefinir senha").

[ ] Geração automática de uma nova página no Confluence em formato de rascunho (Draft) com a estrutura do problema mapeado.

[ ] Notificação via Slack/E-mail para o time de documentação revisar e publicar.
